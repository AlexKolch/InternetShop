# onlineShop
Выполненные задачи: 
1. Реализовать главный экран для приложения онлайн магазина
2. Создать баннер актуальных предложений
3. Реализовать планку с категориями и блок меню
 
 ---
 Использовал в проекте:

* Clean Swift architecture
* UITableView
* UICollectionView
* AutoLayout
* URLSession
* URLCash
* GCD
